# dotfiles

1. kaoriya-vimをダウンロード/任意の場所に解凍
1. msys-gitをインストール
1. git-bashを開き以下のコマンドを実行
```
curl https://raw.githubusercontent.com/Shougo/neobundle.vim/master/bin/install.sh > install.sh
sh /install.sh
```
1. _vimrcおよび_gvimrcをホームディレクトリに配置
1. vimXX/switches/catalog/utf-8.vimをvimXX/switches/enbaledにコピー
1. vimXX/plugins/vimproc/lib/vimproc_win64.dllを$HOME/.vim/bundle/vimproc.vim/lib/にコピー
